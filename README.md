# Derri-Ann Taylor

Author website for Derri-Ann Taylor, poet and author of "Straight From My Heart".

Date: April 2026

Mobile-first responsive design. Screen reader accessible (WCAG 2.1 compliant).

Designed by [Wanda McCrae](https://wandamccrae.com/).

Live site: TBD

## Tech Stack
* HTML
* Tailwind CSS
* Google Fonts

## Accessibility

* Semantic HTML landmarks (`<header>`, `<main>`, `<section>`, `<nav>`, `<footer>`)
* Skip-to-main-content link for keyboard users
* Descriptive alt text and ARIA labels on all links and images
* Visible focus styles for keyboard navigation
* Proper heading hierarchy (h1 → h2)
